# 4-DOF-Robot-Arm-
4 DOF Robot Arm Controlled by Joysticks and 4 Stepper Motors.

A Simple Functional Robotic Arm Design.
Nema 17 Stepper Motors were used with DVR8825 Stepper Motor Drivers.
